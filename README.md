## Principles

* **Fearless**
  - No data races, No unsafe, Minimize unhandled errors

* **High Performance**
  - Everything is Parallelism

* **High Scalability**
  - Everything is Distributed

* **High Reliability**
  - Datafuse primary design goal is reliability
