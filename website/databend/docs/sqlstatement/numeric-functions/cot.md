title: COT
---

Returns the cotangent of x, where x is given in radians.

## Syntax

```sql
COT(x)
```

## Arguments

| Arguments   | Description |
| ----------- | ----------- |
| x | The angle, in radians. |

## Return Type

A Float64 data type value.


## Examples

```
mysql> SELECT COT(12);
+---------------------+
| COT(12)             |
+---------------------+
| -1.5726734063976895 |
+---------------------+
1 row in set (0.00 sec)
```
