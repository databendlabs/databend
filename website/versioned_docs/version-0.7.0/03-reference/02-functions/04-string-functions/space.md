---
title: SPACE
---

Returns a string consisting of N space characters.

## Syntax

```sql
SPACE(n);
```

## Arguments

| Arguments   | Description |
| ----------- | ----------- |
| n | The number of spaces |

## Return Type

String data type value.

## Examples

```txt
SELECT SPACE(20)
+----------------------+
| SPACE(20)            |
+----------------------+
|                      |
+----------------------+
```
