---
title: GEN_RANDOM_UUID
---

Generate a random UUID based on v4.

## Syntax

```sql
GEN_RANDOM_UUID()
```

## Return Type

Random UUID.

## Examples

```sql
mysql> SELECT gen_random_uuid();
+--------------------------------------+
| gen_random_uuid()                    |
+--------------------------------------+
| ab1bce12-4508-4d11-bd96-c42e9e7eefdd |
+--------------------------------------+
```
